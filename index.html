Mis Proyectos

Proyecto 1

Este proyecto, desarrollado en Python, simula el funcionamiento de un ransomware mediante el uso de la biblioteca cryptography. Implementa un sistema de cifrado y descifrado de archivos utilizando el algoritmo Fernet, asegurando la protección de datos mediante una clave secreta. El programa de cifrado busca archivos específicos, los encripta y guarda la clave en un archivo separado. Luego, el programa de descifrado permite restaurar los archivos a su estado original utilizando la clave almacenada. Este proyecto tiene fines educativos y está diseñado para demostrar cómo operan estos tipos de malware(No se asusten mis parceros)
